> **_subject: 프로그래밍 언어의 주요 개념_**

프로그래밍 언어에는 몇 가지 기본 아이디어를 공유한다.

프로그래밍 언어가 입력 데이터를 읽고, 산술 연산을 거쳐서 중간 값을 저장하고, 이 값을 기반으로 결과를 표시한다.

프로그래밍 언어는 위에 기본 아이디어 한 가지 외에 두 가지 아이디어가 더 있다.

# 50 day

> **_subject: 파이썬으로 그래프 그리기_**  
> **_summary: matplotlib.pyplot 라이브러리를 사용하여 파이썬으로 그래프를 그릴 수 있다._**  


```python
import math
import matplotlib.pyploy as plt 

log = []
linear = []
nlogn = []
quadratic = []
for n in range(1, 21):
    linear.append(n)
    log.append(math.log(n))
    nlogn.append(n + math.log(n))
    quadratic.append(n ** 2)
plt.plot(linear, label = 'N')
plt.plot(log, label = 'log N')
plt.plot(log, label = 'N log N')
plt.plot(quadratic[0:10], label='N ** 2')
plt.legend()
plt.show()
```
